Loops

Arrays

Er zijn enkele varianten van de for-lus:

De foreach(achtige) lus

for my $val (@array){
	# doe iets met $val
}
foreach my $val (@array){
	# doe iets met $val
}
for (@array){
	# doe iets met $_
}

Volgens de indices

for my $i (0..$#num){
	say "$i = $num[$i]";
}

Hashes

while (my ($key,$val) = each %hash) {
	say "$key = $val";
}
