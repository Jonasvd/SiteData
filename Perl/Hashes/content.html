Hashes

Aanmaken en opvullen

Een hash kan op enkele manieren opgevuld worden. 
my %hash;
my %hash = (
  naam => 'Jan',
  leeftijd => 21,
  stad => 'Gent'
  );

say "$hash{naam} is $hash{leeftijd} en komt uit $hash{stad}";
Een nieuw paar toevoegen aan een bestaande hash: 
$hash{straat} = 'kerkstraat';
Met een array als keys en een array als values: 
my @valueMap = (1,2,4,8);
my @value = ('een', 'twee', 'vier', 'acht');

my %map;
@map{@valueMap} = @value;

Paar verwijderen

Een key/value paar kan verwijderd worden met delete. 
delete $hash{key}

Slice

Je kunt een slice nemen van enkele waarden en die in een array bewaren: 
my @adres = @hash{'straat', 'stad'};
say join " ", @adres;

Naar array omvormen

Als je een hash in een array steekt krijg je een array met als inhoud { key1, value1, key2, value2, key3, value3 }. 
my @array = %hash;

Overlopen

keys en values

Een array met de keys of values kan opgevuld worden. 
my @keys = keys %hash;
my @values = values %hash;

say "keys: ", join ", ", @keys;
say "values: ", join ", ", @values;

Als key/value pair

Met each kun je over alle paren lopen. Typisch gebruik je dat in een while-lus. 
while (my ($key, $value) = each %hash){
  say "$key : $value";
}
